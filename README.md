## :books: Información general

#Funcionamiento
* Dentro de la app el usuario puede:
* Registrarse
* Iniciar sesión
* Configurar perfil (foto de perfil)
* Ver usuarios registrados.
* Ver chats.
* Enviar y recibir mensajes.
* Enviar y recibir emojis.
* Cerrar sesión
